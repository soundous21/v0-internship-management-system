## الأدوار والأذونات

### Student (ROLE_STUDENT)
```php
#[IsGranted('ROLE_STUDENT')]
class StudentController
```
- البحث عن الفرص
- تقديم الطلبات
- إدارة الملف الشخصي

### Company (ROLE_COMPANY)
```php
#[IsGranted('ROLE_COMPANY')]
class CompanyController
```
- نشر الفرص
- إدارة المتقدمين
- قبول/رفض الطلاب

### Admin (ROLE_ADMIN)
```php
#[IsGranted('ROLE_ADMIN')]
class AdminController
```
- التحقق من الحسابات
- إدارة النظام
- عرض الإحصائيات
